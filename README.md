# image-upload-nodejs
image upload using nodejs ejs and multer

An example of how to upload images locally with Node.js and Multer.

### Version
1.0.0

## Usage

### Installation

Install the dependencies

```sh
$ npm i
```

### Serve
To serve in the browser

```sh
$ npm start
```
